# notas-atletas
Notas dos Atletas

Este projeto foi desenvolvido como solução para um desafio de lógica de programação, utilizando JavaScript puro.
O objetivo é calcular a média válida das notas de atletas em uma competição de ginástica artística, seguindo regras específicas.

Descrição do Desafio

Cada atleta recebe cinco notas de jurados diferentes.
Para calcular a média válida, devem ser consideradas apenas as três notas centrais, descartando:

a maior nota, e

a menor nota.

A aplicação deve:

✔ Receber uma lista de atletas e suas notas
✔ Ordenar as notas
✔ Eliminar a maior e a menor
✔ Calcular a média das três restantes
✔ Exibir o resultado para cada atleta
